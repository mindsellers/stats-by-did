Скрипт считает количество вызовов за указанный промежуток времени, поступивший на каждый номер(did) из списка, который получает из файла numlist, расположенный в директории, где лежит сам скрипт и из которой мы его запускаем.
Скрипт выбирает вызовы по did, статусу disposition ANSWERED и по признаку, что последнее приложение lastapp не равно Playback, то есть отсекает вызовы, где клиент или скинул трубку в процессе приветствия, или звонил в нерабочее время и попал на ночное уведомление
В скрипте необходимо задать кредлы к мускулю # Параметры подключения (измените под свои) и промежуток за который выбираем звонки # Начало и конец интервала
